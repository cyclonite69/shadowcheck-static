üöÄ Quick Start

Prerequisites

  • Node.js 18+
  • PostgreSQL 18+ with PostGIS extension
  • Git

Installation

git clone https://github.com/cyclonite69/shadowcheck-static.git cd shadowcheck-static npm install

Database Setup

createdb shadowcheck psql -d shadowcheck -c "CREATE EXTENSION postgis;"

Configuration

cp .env.example .env # Edit .env with your database credentials DB_USER=shadowcheck DB_HOST=localhost DB_NAME=shadowcheck DB_PASSWORD=your_password DB_PORT=5432

Start Server

npm start # Server runs on http://localhost:3001
